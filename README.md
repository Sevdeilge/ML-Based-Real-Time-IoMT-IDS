# Machine Learning Based Real-Time Intrusion Detection System for IoMT Networks using FastAPI

## Overview

This project presents a machine learning-based real-time Intrusion Detection System (IDS) designed for Internet of Medical Things (IoMT) networks.

The system analyzes network traffic, extracts statistical features, and classifies malicious activities using a trained machine learning model.

## System Architecture

The system consists of:

- CIC IoMT Dataset
- Machine Learning Model
- FastAPI Prediction Service
- Streamlit SOC Dashboard
- Real-time Traffic Capture using Tshark

## Technologies

- Python
- Scikit-Learn
- FastAPI
- Streamlit
- Tshark
- Redis

## Features

- Real-time network traffic analysis
- Attack classification
- REST API based prediction service
- Security monitoring dashboard
